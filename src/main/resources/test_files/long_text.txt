去了趟男士高端会所，出来后看不上自己老婆

Original beebee  beebee星球  2022-04-14 11:18
当一个场地冠以会所之名，那它必然不再下里巴人；当会所加上高端两字作为定语，那就绝非普通群众可以涉足之地。

当高端两字前再加上男士作为定义，它就成了某种社会身份的象征。

这是个贩夫走卒难以跨越的门槛，一旦迈入那个镶金带紫的大厅，你就不再是一个人，而是一种叫贵宾的高级存在。

“人并不值钱，就算在经济最发达的城市。”王建老师朝我略微摇了摇手指，我谄媚地给他把烟插鼻孔里点燃，就像高中时那样。

一日为师，终生为父。
Image

我高中时和你讲过维特根斯坦，你需要明白语句与语义、表达与本质、意象与应用、修辞与经验之间真正的关系。

还记得我叫你摘抄的那句话吗：我要教你的是从不明显的胡说过渡到明显的胡说。

“记得。”我说，维特根斯坦《哲学研究》第464节。

“给出一个预期，那么无论接下来发生了什么，都必定与它一致或不一致——除非对预期的表达是不准确的。《哲学研究》第465节。”

那你的预期一致了吗？王建老师问我。

“绝对一致。”

我看了看出来后业务经理递给我的卡片，觉得食髓知味。
Image

王建老师从中山装左胸口袋里掏出一叠另外的卡片，对我说，纷繁表象下其实藏着显而易见的本质，但没有人愿意真正去说破它，因为说破代表着冒犯，而冒犯则代表受到惩罚。

即便冒犯和惩罚只是一种可能性、一种弥漫而潜藏的威胁、一种规则所暗示的强制。

趋利避害是人的本能。

你看看我的卡片，这是我收集的部分普通卡片，有点像蟑螂，一般从宾馆的门缝溜进来，或者散落在大排档带着虾壳的油腻水泥地上。卡片上的言语轻佻，图片浮夸，做工低劣，容易褪色。碰一下，画中女郎的连衣裙，就要沾到你的大拇指上，洗不掉，还渗出一股汗臭。
Image

而你手里的卡片呢？

精致、刻意、带有磨砂质感，还有一股芳香，说不清是什么味道，像是茉莉花，又像是熏香。每当你走进领导的办公室，或是去高迁的亲戚家拜年，你就能在他们身上闻见这种味道。
Image

这些卡片透着一股华贵与尊崇，印刻着皇冠与权杖，没有乱七八糟的女性照片，免去了版权之忧。它完全脱离了低级趣味，一味向上兼容，符合顶层设计和高层眼界，就像在下一步大棋。

好这一口的，最差也得是个路虎车主，车上放着一盘邓丽君的CD，后备箱有五粮液跟国窖1573，旁边脏蜜的腿都要比老百姓的白。
Image

从卡片上可以看出来，会所的名字，通常带有龙，凤，或者是君，爵等隐喻权力的字眼。一看就很高端，一看你就会明白，会所里面肯定有一些庙堂之上的项目。

了无分文的人，通常都进不了它的门。你不自量力走进去，要是手腕上没有劳力士，迎宾少爷多半以为你是暗访的记者。他会指一指天花板，然后说他们上面有人，让你做完服务就快点离开。
Image

卡片的主色调，一律是以暗金，黄紫，黑曜石等旧社会象征九五之尊的忌讳色调为主。如果卡片会唱歌，那么一定是在唱向天再借五百年。

普通的男人扛不住这种设计，有压迫感，想跪下磕头。电子厂的兄弟要是走进去，就像是去开厂里的表彰动员大会，又害怕又浑身不自在，除了感恩什么都说不出口。

但电子厂的厂长一般都很爱去，还爱办理VIP。会员卡要是被妻子发现了，就指指上面，压低语气，说不收不行。
Image

高端男士会所的卡片上，没有任何多余的信息。只有地址，会所名称，以及某位经理的联系方式，很隐晦，不知道会所的服务内容，既贯彻了保密原则，又透露了关键信息。像知名企业家的自传，说一半，留一半。

你打电话给孙经理，孙经理一听你声音，就说，X总，马上安排，然后挂了电话，就像是你的秘书。一切都在不言中，行稳致远。
Image

“人往高处走，想要出人头地，总要踏出这一步，谁也不愿意一直玩烂的，在泥沼里逐浪。”我问王建老师：“那么，怎么突围呢？”

你已经突围了。王建老师淡然一笑，用中指堵住一边鼻孔，然后猛烈喷气把烟头射出。我赶紧给他续上一根，点头哈腰，像一位基层干部。

高端男士会所，往往是熟人引荐。

比如，你晚上跟领导出去应酬，都喝多了，领导拍拍你的肩，说自己突然有点不服老，想挑战一下自己的软肋，然后就会丢给你一张会所的名片，让你去安排。

这种名片很低调，不浮夸，但拿在手里，能触摸到一股傲气，它也许被龙脉温润过。有帝王之相，难怪领导会喜欢。

然后，你就成了其中的一份子。
Image

当然，也有的卡片，直接出自会所经理之手。

经理一般跟你老同学走得很近，老同学搞地产发了家。经理都很壮，很高，还很帅，开车还戴着白手套，还有眼镜，应该是本科毕业，认识很多大人物。经理散给你一张名片，你用双手接住。仿佛接住的不是卡片，是人脉。

你知道这经理只是奴才，但你有求于他，所以你必须把奴才当领导。

你心想儿子以后读重点高中，这位经理应该能帮上忙。

又或者，当自己成为VIP，那经理就会成为你的奴才。叫你哥。
Image

“我明白了。”

我向王建老师磕头，感恩，然后和他讲了个上个月的故事。

那次，跟一个在医药集团当主任的朋友吃饭，叙旧，搞到晚上十一点。

吃完饭，他说，不如去会所玩一玩，高端的那种。我问他，消费高吗？他顿了一下，反问我，你没有事情要让我帮忙吗？我说没有，只是叙旧，一起骂学校跟老师。他没再说话，瞥了我一眼，就打车走了。
